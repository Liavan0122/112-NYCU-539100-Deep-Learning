## 環境建設要求

### Software Requirements

* Python 3.6.1+
* gcc 4.9+ for PyTorch1.0.0+
* Cuda 8.0, 9.0, 9.1, 10.0 depending on each DNN library
* Cudnn 6+, 7+
* NCCL 2.0+ (for the use of multi-GPUs)
